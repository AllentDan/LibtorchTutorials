# TeachLibtorch

# Description
This is a code repository for libtorch tutorials, which contains lessons for utilizing libtorh (or pytorch c++).

# Environment
- Win10
- Libtorch 1.7
- Opencv 4.5
- CUDA 10.2 && CUDNN
- Qt creator or Visual studio
